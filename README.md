# Portfolio Web

## Descripción del proyecto
Este proyecto es un portafolio web personal diseñado para presentar mis habilidades como desarrolladora web junior. El sitio incluye información sobre un perfil profesional, las tecnologías que utilizo y algunos de sus proyectos destacados.

El objetivo de este proyecto es mostrar mi capacidad para crear interfaces web limpias, responsivas y organizadas siguiendo buenas prácticas de desarrollo.

## Funcionalidad
El sitio web permite:

- Mostrar información personal y profesional.
- Visualizar habilidades y herramientas utilizadas en desarrollo web.
- Presentar proyectos destacados con una breve descripción.
- Acceder a enlaces para ver el proyecto en vivo o revisar el código en GitHub.
- Contactar a través de enlaces a redes sociales o correo electrónico.

## Tecnologías y técnicas utilizadas

Tecnologías:

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Responsive Design
- Git
- GitHub
- GitHub Pages

Técnicas utilizadas:

- Maquetación con Flexbox y Grid
- Organización semántica del HTML
- Diseño responsivo mediante media queries
- Uso de imágenes y SVG para iconos
- Control de versiones con Git
